# Task List Pengembangan Aplikasi JasaCepat

## 1. Persiapan Project
- [ ] Menyiapkan environment development Flutter
- [ ] Memastikan semua dependency terinstal dengan benar
- [ ] Mengonfigurasi emulator/device untuk testing
- [ ] Meninjau struktur folder proyek

## 2. UI/UX Dasar
- [ ] Membuat splash screen yang menarik dan profesional
- [ ] Membuat halaman login yang rapi dan mudah dipakai
- [ ] Membuat halaman utama dengan daftar layanan
- [ ] Membuat halaman profil pengguna
- [ ] Membuat halaman pesanan pengguna
- [ ] Membuat halaman chat sederhana

## 3. Fitur Utama Aplikasi
- [ ] Menyusun alur login pengguna
- [ ] Menampilkan daftar layanan berdasarkan kategori
- [ ] Menampilkan teknisi terdekat di area pengguna
- [ ] Menampilkan peta lokasi teknisi
- [ ] Menyediakan tombol aksi untuk memilih layanan
- [ ] Menyediakan halaman detail layanan/teknisi

## 4. Fitur Admin
- [ ] Membuat dashboard admin
- [ ] Menampilkan statistik utama admin
- [ ] Menampilkan aktivitas pesanan terbaru
- [ ] Menyediakan navigasi logout admin

## 5. Navigasi dan State Management
- [ ] Mengatur routing antar halaman
- [ ] Menjaga navigasi antar screen tetap konsisten
- [ ] Mengelola state untuk halaman utama dan profil
- [ ] Menyederhanakan alur aplikasi agar lebih intuitif

## 6. Integrasi Data
- [ ] Menyiapkan data dummy untuk layanan
- [ ] Menyiapkan data dummy teknisi
- [ ] Menyiapkan data dummy pesanan
- [ ] Menyiapkan data admin demo

## 7. Testing dan Validasi
- [ ] Menguji aplikasi pada emulator/device
- [ ] Memastikan login berjalan dengan benar
- [ ] Memastikan navigasi antar halaman berjalan lancar
- [ ] Memastikan aplikasi tidak crash saat dipakai
- [ ] Melakukan pengecekan UI pada layar kecil dan besar

## 8. Perbaikan dan Optimasi
- [ ] Menghapus bug yang ditemukan selama testing
- [ ] Memperbaiki tampilan agar lebih konsisten
- [ ] Mengoptimalkan performa aplikasi
- [ ] Membersihkan code yang tidak terpakai

## 9. Rilis Awal (MVP)
- [ ] Menyusun build APK debug
- [ ] Melakukan testing final
- [ ] Menyiapkan dokumentasi singkat aplikasi
- [ ] Siap untuk demo atau presentasi
